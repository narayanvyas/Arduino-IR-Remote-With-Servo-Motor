# IR Remote With Servo Motor
Arduino IR Remote With Servo Motor.

## Prerequisites
1. Arduino Uno
2. IR Receiver Module
3. IR Remote
4. Jumper Wires
5. Breadboard / PCB
6. Servo Motor
7. 2 LEDs
8. 2 x 220 Ohm Resistors
